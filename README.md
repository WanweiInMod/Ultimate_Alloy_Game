# 究极合金

欢迎来到究极合金！

这是一个以“挖掘矿物、制作合金”为主题的增量游戏。

点击挖矿，搭建熔炼炉，招募矿工，最后造出究极合金！

---

核心玩法：通过初期的点击获得矿石，并熔炼金属锭，通过合金窑获取各种合金；

随着不同种类材料逐渐发现，研究解锁更多矿物的获取，招募矿工自动化原矿获取；

最终获得独一无二的究极合金。

---

第一阶段：

目标：引导玩家了解最基础的6项资源

1. 点击挖矿按钮，获取石头
2. 使用石头购买熔炉，熔炉消耗石头，并随机产出金属（金银铜铁）
3. 解锁交易行的“贸易”标签，出售金属获取资金
4. 使用资金招募矿工，矿工自动生产石头

---

第二阶段：

目标：引导玩家了解资源的更多获取机制，强化资源循环

1. 使用资金建立“行动部”与“研发部”，用于提供限时与永久的加成机制，如矿物生产概率累加等
2. 通过“研发部”解锁“珍稀矿材”资源（稀有性类似于cookie clicker中的黄金饼干）的发现能力，玩家可在点击挖矿时收获到
3. 获得任意“珍稀矿材”后解锁交易行的“珍品拍卖”功能。
（“珍稀矿材”收获时，以及其在交易行售出时，提供与“行动部”类似的限时加成机制）
4. 通过“行动部”与“研发部”逐渐解锁资源增产的机制
5. 通过“研发部”解锁采矿机，提高点击时的石头增益

额外：解锁到一定条目后，挖矿将概率产出“未知矿物”

---

第三阶段：

目标：引导玩家了解游戏的核心机制：“合金”

1. 在通过“行动部”与“研发部”解锁资源增产机制达到某一阶段后，会抵达上阶段设定的生产概率上限。
此时出现需要满足一定生产次数才可解锁的“研究项”，在玩家达到该要求后，通过该词条的研究解锁“合金”
2. 解锁的“合金”可以是任意一种基础金属（金银铜铁）的双金属合金，例如青铜、黄铜等。
当前的任何合金暂时只可通过熔炉，作为该金属的概率产物产出
3. 通过“行动部”解锁出售合金的贸易词条
4. 玩家再次推进“行动部”与“研发部”的机制解锁，直到解锁特定条目

---

第四阶段（主要循环玩法）：

目标：使玩家自主可控地产出合金，并尝试发现更多种类的基础金属资源（锡、锌、铅、镍等）

1. 解锁“高炉”“合金窑”：
“高炉”可制造钢等高温要求合金，“合金窑”可制造一般要求下的合金
2. 玩家在发现一种合金的所有需求基础材料后，
在该合金从熔炉生产一定次数或玩家主动尝试了解的配方（例如铜类合金的比例等）后则解锁对应的量产配方
3. 根据需要产出的合金，分配熔炉的矿物产出比例（需要前置机制解锁）并量产出售
